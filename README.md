# 🏎️ My Passionate Car Database
 
## 概要
このプロジェクトは、自身の最大のパッションである「スポーツカー」のスペックを管理するためのMySQLデータベースです。
専門学校で学んでいるIT技術を使い、単なる知識ではなく「動く形」としてアウトプットしました。
 
## 収録済みの名車リスト
- **Honda S2000 (F20C)** - 9,000回転まで回るVTECの至宝。NAの魅力を追求。
- **Nissan Skyline GT-R (R34)** - RB26DETTを搭載した最強の直6ターボ。
- **Mazda RX-7 (FD3S)** - 唯一無二のロータリーエンジン搭載機。
 
## 技術的こだわり
- **データ設計**: 単なる車種名だけでなく、`aspiration`（吸気方式：NA/Turbo/SC）のカラムを設け、スポーツカーの特性を技術的に分類。
- **環境**: MacBook Pro / MySQL 5.7 / Homebrew を使用して環境構築から実施。
